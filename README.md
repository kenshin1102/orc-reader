## OCR reader with TessaractOCR
Read string from images/pdf

## Requirements
- python >= 3.6

## Install
- get packages

```shell script
make install
```

## How to use

```shell script
poetry run  python img_to_text.py --input=/var/www/python/orc-reader/orc_reader/datatest/img-test-2.png --verbose --output=test.log
```

## Code convention

```shell script
make fix
```
